<b>Подготовка:</b>
- В файле src/main.py подставить ссылку

<b>Запустить скрипт:</b>

```bash
# Create a virtual env
$ python -m venv .venv

# Activate a virtual env
$ source .venv/bin/activate

# Install dependencies
$ pip install poetry

$ poetry install --no-root

# Run the project
$ poetry run python ./src/main.py
```
